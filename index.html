8 Sanity Checks — Specific to v5

⚖️

✓ Class Imbalance — CLASS_WEIGHT applied

Your v5 uses CLASS_WEIGHT={0:0.632, 1:2.388} via N/(2·N_c). Never report accuracy alone — always lead with AUC + sensitivity.

🚰

✓ No Data Leakage — StandardScaler inside Pipeline

Scaler is fitted on X_train only per fold via Pipeline([('scaler',...),('svc',...)]). rand_search.fit(X_train,...) is correct.

🔀

✓ Stratified Splits — StratifiedKFold both loops

Both inner and outer preserve the Brugada/Normal ratio (~15 Brugada per fold). Plain KFold could produce folds with 0 Brugada and crash roc_auc_score.

🔁

✓ Reproducibility — All seeds fixed

outer random_state=42, inner random_state=0, RandomizedSearchCV random_state=42, SVC random_state=42. Two runs = identical results.

🔢

⚠ NaN/Inf Check — Add after feature loop

Add: assert not np.isnan(X).any() and assert not np.isinf(X).any() right after building X to catch boundary R-peak edge cases.

📐

⚠ Clinical Direction Check — Read Step 4 output

Your code prints PASS ✓ / FAIL for x_1 to x_4. All four must PASS on real data. Any FAIL means the feature window is wrong for that signal.

🎯

⚠ Sensitivity Target — must be ≥ 0.85

If sensitivity is low, lower decision threshold: y_pred = (y_prob >= 0.35).astype(int) instead of the default 0.5.

🔄

ℹ AUC Inversion Fix — watch fold count

One flip per run is normal. If 3+ folds flip, C is hitting the 0.01 lower bound — raise loguniform(0.1, 1e4). Check printed best_C values.
